合併排序法演示

1秒

演算法原理

合併排序法（Merge Sort）是一種基於分治法的排序演算法。它將陣列分成兩個子陣列，遞歸地對每個子陣列進行排序，然後將兩個已排序的子陣列合併成一個有序的陣列。

這個演算法的關鍵在於合併步驟，它將兩個已排序的子陣列合併成一個有序的陣列。

偽代碼

1 procedure mergeSort(A, left, right)
2 if left < right then
3 mid = (left + right) / 2
4 mergeSort(A, left, mid)
5 mergeSort(A, mid+1, right)
6 merge(A, left, mid, right)
7 end if
8 end procedure
9 procedure merge(A, left, mid, right)
10 n1 = mid - left + 1
11 n2 = right - mid
12 let L[0..n1-1] be new array
13 let R[0..n2-1] be new array
14 for i = 0 to n1-1 do
15 L[i] = A[left + i]
16 end for
17 for j = 0 to n2-1 do
18 R[j] = A[mid + 1 + j]
19 end for
20 i = 0, j = 0, k = left
21 while i < n1 and j < n2 do
22 if L[i] <= R[j] then
23 A[k] = L[i]
24 i = i + 1
25 else
26 A[k] = R[j]
27 j = j + 1
28 end if
29 k = k + 1
30 end while
31 while i < n1 do
32 A[k] = L[i]
33 i = i + 1
34 k = k + 1
35 end while
36 while j < n2 do
37 A[k] = R[j]
38 j = j + 1
39 k = k + 1
40 end while
41 end procedure

複雜度分析

  • 時間複雜度：
    • 最佳情況：O(n log n)
    • 最壞情況：O(n log n)
    • 平均情況：O(n log n)
  • 空間複雜度：O(n) - 需要額外的空間來存儲臨時陣列
  • 穩定性：穩定排序 - 相等元素的相對順序不會改變
